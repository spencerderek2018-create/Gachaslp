## Overview

This document covers how to build the game system by system, starting with the combat prototype. All code is C# for Unity. Systems are built in dependency order — later systems depend on earlier ones.

**Build Order:**
1. Data Models (foundation everything else sits on)
2. Element & Debuff System
3. Combat Readiness & Turn Manager
4. Battle Unit (runtime hero)
5. Skill System
6. Soul System
7. Dual Attack
8. Battle Manager (orchestrator)
9. Equipment System
10. Game Mode Architecture (post-prototype)

## Phase 2: Element & Debuff System

### ElementType.cs
```csharp
public enum ElementType { Fire, Water, Earth, Light, Dark }

public static class ElementalAffinity
{
    // Returns the damage/debuff multiplier for attacker element vs defender element
    // 1.15f = advantage, 0.85f = disadvantage, 1.0f = neutral
    private static readonly float[,] affinityMatrix = new float[5, 5]
    {
        //          Fire   Water  Earth  Light  Dark   (defender)
        /* Fire  */ { 1.0f, 0.85f, 1.15f, 1.0f,  1.0f  },
        /* Water */ { 1.15f, 1.0f, 0.85f, 1.0f,  1.0f  },
        /* Earth */ { 0.85f, 1.15f, 1.0f, 1.0f,  1.0f  },
        /* Light */ { 1.0f,  1.0f,  1.0f, 1.0f,  1.15f },
        /* Dark  */ { 1.0f,  1.0f,  1.0f, 1.15f, 1.0f  },
    };

    public static float GetMultiplier(ElementType attacker, ElementType defender)
    {
        return affinityMatrix[(int)attacker, (int)defender];
    }

    public static bool HasAdvantage(ElementType attacker, ElementType defender)
    {
        return GetMultiplier(attacker, defender) > 1.0f;
    }

    // Debuff landing chance bonus when attacker has elemental advantage
    public const float DebuffChanceBonus = 0.15f;
}
```

### DebuffSystem.cs
```csharp
using System.Collections.Generic;
using UnityEngine;

public enum DebuffType
{
    None,
    // Fire debuffs — Damage over Time
    Burn,       // deals ATK-based damage each turn
    Scorch,     // stacking burn; each stack increases DoT damage
    Ignite,     // burn that also reduces healing received
    // Dark debuffs — Stat Reduction / Weakening
    Curse,      // reduces ATK by X%
    Blind,      // reduces Crit Rate by X%
    Weaken,     // reduces all damage dealt by X%
    Silence,    // prevents S2/S3 use (only S1 allowed)
    // Water debuffs — CR Reduction
    Chill,      // reduces CR by X% on application
    Freeze,     // skips unit's next turn
    // Earth debuffs — Control
    Stun,       // skips turn
    Provoke,    // forces attack on a specific target
}

public static class DebuffCategory
{
    public static bool IsDoT(DebuffType type) =>
        type == DebuffType.Burn || type == DebuffType.Scorch || type == DebuffType.Ignite;

    public static bool IsStatReduction(DebuffType type) =>
        type == DebuffType.Curse || type == DebuffType.Blind ||
        type == DebuffType.Weaken || type == DebuffType.Silence;

    public static bool IsCRReduction(DebuffType type) =>
        type == DebuffType.Chill || type == DebuffType.Freeze;

    public static bool IsControl(DebuffType type) =>
        type == DebuffType.Stun || type == DebuffType.Provoke;

    // Fire debuffs are stronger vs Earth. Dark debuffs are stronger vs Light.
    public static ElementType GetDebuffElement(DebuffType type)
    {
        if (IsDoT(type))         return ElementType.Fire;
        if (IsStatReduction(type)) return ElementType.Dark;
        if (IsCRReduction(type)) return ElementType.Water;
        if (IsControl(type))     return ElementType.Earth;
        return ElementType.Fire;
    }
}

public class DebuffInstance
{
    public DebuffType type;
    public int remainingTurns;
    public float magnitude;      // e.g. 0.2 = 20% stat reduction, or flat DoT damage
    public BattleUnit source;    // who applied it (for scaling off their ATK)

    public DebuffInstance(DebuffType type, int duration, float magnitude, BattleUnit source)
    {
        this.type = type;
        this.remainingTurns = duration;
        this.magnitude = magnitude;
        this.source = source;
    }

    // Called at the start of the afflicted unit's turn
    public void OnTurnStart(BattleUnit afflicted)
    {
        if (DebuffCategory.IsDoT(type))
        {
            float dotDamage = source.FinalATK * magnitude;
            if (type == DebuffType.Scorch)
                dotDamage *= afflicted.GetStackCount(DebuffType.Scorch);
            afflicted.TakeDamage(Mathf.RoundToInt(dotDamage), isDot: true);
        }
        remainingTurns--;
    }

    public bool IsExpired => remainingTurns <= 0;
}
```

## Phase 3: Combat Readiness & Turn Manager

The CR system is the heart of the game. Every tick, all units fill their CR gauge based on Speed. The unit that hits 100 first takes their turn.

### TurnManager.cs
```csharp
using System.Collections.Generic;
using System.Linq;
using UnityEngine;

public class TurnManager
{
    private List<BattleUnit> allUnits;

    public TurnManager(List<BattleUnit> units)
    {
        allUnits = units;
    }

    // Advance CR gauges until one unit hits 100. Returns that unit.
    public BattleUnit AdvanceToNextTurn()
    {
        while (true)
        {
            BattleUnit ready = allUnits
                .Where(u => !u.IsDead)
                .FirstOrDefault(u => u.CombatReadiness >= 100f);

            if (ready != null)
            {
                ready.CombatReadiness -= 100f;
                return ready;
            }

            TickAllUnits();
        }
    }

    private void TickAllUnits()
    {
        // Find smallest CR gap so we advance exactly to 100 for the fastest unit
        float minGap = allUnits
            .Where(u => !u.IsDead)
            .Min(u => (100f - u.CombatReadiness) / u.FinalSpeed);

        foreach (var unit in allUnits.Where(u => !u.IsDead))
            unit.CombatReadiness += unit.FinalSpeed * minGap;
    }

    // Push a unit's CR forward (e.g. a support skill)
    public void PushCR(BattleUnit target, float amount)
    {
        target.CombatReadiness = Mathf.Min(100f, target.CombatReadiness + amount);
    }

    // Pull a unit's CR back (e.g. Water elemental debuff)
    public void PullCR(BattleUnit target, float amount)
    {
        target.CombatReadiness = Mathf.Max(0f, target.CombatReadiness - amount);
    }

    // Returns the projected turn order for UI display (next N turns)
    public List<BattleUnit> PreviewTurnOrder(int count)
    {
        // Clone CR values for simulation
        var snapshot = allUnits
            .Where(u => !u.IsDead)
            .Select(u => (unit: u, cr: u.CombatReadiness))
            .ToList();

        var order = new List<BattleUnit>();
        for (int i = 0; i < count; i++)
        {
            while (true)
            {
                var ready = snapshot.FirstOrDefault(s => s.cr >= 100f);
                if (ready.unit != null)
                {
                    order.Add(ready.unit);
                    int idx = snapshot.IndexOf(ready);
                    snapshot[idx] = (ready.unit, ready.cr - 100f);
                    break;
                }
                float minGap = snapshot.Min(s => (100f - s.cr) / s.unit.FinalSpeed);
                for (int j = 0; j < snapshot.Count; j++)
                {
                    var s = snapshot[j];
                    snapshot[j] = (s.unit, s.cr + s.unit.FinalSpeed * minGap);
                }
            }
        }
        return order;
    }
}
```

## Phase 4: Battle Unit (Runtime Hero)

`BattleUnit` is the live, in-battle representation of a hero. It wraps `HeroData` and tracks mutable runtime state (current HP, CR, debuffs, cooldowns).

### BattleUnit.cs
```csharp
using System.Collections.Generic;
using System.Linq;
using UnityEngine;

public class BattleUnit
{
    public HeroData Data { get; private set; }
    public bool IsPlayerTeam { get; private set; }

    // Runtime state
    public float CombatReadiness { get; set; }
    public int CurrentHP { get; private set; }
    public bool IsDead => CurrentHP <= 0;

    private List<DebuffInstance> activeDebuffs = new();
    private Dictionary<SkillSlot, int> cooldowns = new();

    // Computed final stats (base + equipment + buffs)
    public int   FinalHP      => ComputeStat(StatType.HP);
    public int   FinalATK     => ComputeStat(StatType.ATK);
    public int   FinalDEF     => ComputeStat(StatType.DEF);
    public int   FinalSpeed   => ComputeStat(StatType.Speed);
    public float FinalCritRate    => ComputeFloatStat(StatType.CritRate);
    public float FinalCritDamage  => ComputeFloatStat(StatType.CritDamage);
    public float FinalEffectiveness => ComputeFloatStat(StatType.Effectiveness);
    public float FinalEffectResist  => ComputeFloatStat(StatType.EffectResist);

    private StatModifiers modifiers = new();

    public BattleUnit(HeroData data, bool isPlayerTeam)
    {
        Data = data;
        IsPlayerTeam = isPlayerTeam;
        CurrentHP = FinalHP;
        CombatReadiness = 0f;

        foreach (SkillSlot slot in System.Enum.GetValues(typeof(SkillSlot)))
            cooldowns[slot] = 0;
    }

    public void TakeDamage(int amount, bool isDot = false)
    {
        // DEF reduces non-DoT damage: damage = amount * (1 - DEF / (DEF + 1000))
        if (!isDot)
        {
            float defReduction = FinalDEF / (FinalDEF + 1000f);
            amount = Mathf.RoundToInt(amount * (1f - defReduction));
        }
        CurrentHP = Mathf.Max(0, CurrentHP - amount);
    }

    public void Heal(int amount)
    {
        // Check if Ignite debuff reduces healing
        float healMult = HasDebuff(DebuffType.Ignite) ? 0.5f : 1f;
        CurrentHP = Mathf.Min(FinalHP, CurrentHP + Mathf.RoundToInt(amount * healMult));
    }

    public bool TryApplyDebuff(DebuffInstance debuff, BattleUnit attacker)
    {
        // Effect resistance check
        float landChance = debuff.source.FinalEffectiveness - FinalEffectResist;

        // Elemental advantage on the debuff type gives +15% land chance
        ElementType debuffElement = DebuffCategory.GetDebuffElement(debuff.type);
        if (ElementalAffinity.HasAdvantage(debuffElement, Data.element))
            landChance += ElementalAffinity.DebuffChanceBonus;

        landChance = Mathf.Clamp01(landChance);
        if (Random.value > landChance) return false;

        activeDebuffs.Add(debuff);
        ApplyStatModifiersFromDebuff(debuff);
        return true;
    }

    public void OnTurnStart()
    {
        // Tick DoT debuffs
        foreach (var debuff in activeDebuffs.ToList())
        {
            debuff.OnTurnStart(this);
            if (debuff.IsExpired)
            {
                RemoveStatModifiersFromDebuff(debuff);
                activeDebuffs.Remove(debuff);
            }
        }

        // Decrement skill cooldowns
        foreach (var slot in cooldowns.Keys.ToList())
            if (cooldowns[slot] > 0) cooldowns[slot]--;
    }

    public bool IsSkillReady(SkillSlot slot) => cooldowns[slot] <= 0;

    public void SetCooldown(SkillSlot slot, int turns)
    {
        cooldowns[slot] = turns;
    }

    public bool HasDebuff(DebuffType type) => activeDebuffs.Any(d => d.type == type);

    public int GetStackCount(DebuffType type) => activeDebuffs.Count(d => d.type == type);

    public bool IsStunned => HasDebuff(DebuffType.Stun) || HasDebuff(DebuffType.Freeze);
    public bool IsSilenced => HasDebuff(DebuffType.Silence);

    private int ComputeStat(StatType stat)
    {
        // Base stat * modifiers from equipment, buffs, debuffs
        float base_ = stat switch
        {
            StatType.HP    => Data.baseHP,
            StatType.ATK   => Data.baseATK,
            StatType.DEF   => Data.baseDEF,
            StatType.Speed => Data.baseSpeed,
            _ => 0
        };
        return Mathf.RoundToInt(base_ * modifiers.GetMultiplier(stat) + modifiers.GetFlat(stat));
    }

    private float ComputeFloatStat(StatType stat)
    {
        float base_ = stat switch
        {
            StatType.CritRate       => Data.baseCritRate,
            StatType.CritDamage     => Data.baseCritDamage,
            StatType.Effectiveness  => Data.baseEffectiveness,
            StatType.EffectResist   => Data.baseEffectResist,
            _ => 0f
        };
        return Mathf.Clamp01(base_ + modifiers.GetFlat(stat));
    }

    private void ApplyStatModifiersFromDebuff(DebuffInstance debuff)
    {
        switch (debuff.type)
        {
            case DebuffType.Curse:   modifiers.AddMultiplier(StatType.ATK,   -debuff.magnitude); break;
            case DebuffType.Blind:   modifiers.AddFlat(StatType.CritRate,    -debuff.magnitude); break;
            case DebuffType.Weaken:  modifiers.AddMultiplier(StatType.ATK,   -debuff.magnitude); break;
        }
    }

    private void RemoveStatModifiersFromDebuff(DebuffInstance debuff)
    {
        switch (debuff.type)
        {
            case DebuffType.Curse:   modifiers.RemoveMultiplier(StatType.ATK,   -debuff.magnitude); break;
            case DebuffType.Blind:   modifiers.RemoveFlat(StatType.CritRate,    -debuff.magnitude); break;
            case DebuffType.Weaken:  modifiers.RemoveMultiplier(StatType.ATK,   -debuff.magnitude); break;
        }
    }
}

public enum StatType { HP, ATK, DEF, Speed, CritRate, CritDamage, Effectiveness, EffectResist }

// Tracks additive multipliers and flat bonuses per stat
public class StatModifiers
{
    private Dictionary<StatType, float> multipliers = new();
    private Dictionary<StatType, float> flats       = new();

    public float GetMultiplier(StatType s) => 1f + multipliers.GetValueOrDefault(s, 0f);
    public float GetFlat(StatType s)       => flats.GetValueOrDefault(s, 0f);

    public void AddMultiplier(StatType s, float v)    => multipliers[s] = multipliers.GetValueOrDefault(s) + v;
    public void RemoveMultiplier(StatType s, float v) => multipliers[s] = multipliers.GetValueOrDefault(s) - v;
    public void AddFlat(StatType s, float v)          => flats[s] = flats.GetValueOrDefault(s) + v;
    public void RemoveFlat(StatType s, float v)       => flats[s] = flats.GetValueOrDefault(s) - v;
}
```

## Phase 5: Skill System

Skills are executed by the `BattleManager`. Each `SkillEffect` is resolved in sequence.

### SkillResolver.cs
```csharp
using System.Collections.Generic;
using System.Linq;
using UnityEngine;

public static class SkillResolver
{
    public static void Execute(
        BattleUnit caster,
        SkillData skill,
        List<BattleUnit> targets,
        List<BattleUnit> allUnits,
        SoulManager soulManager,
        TurnManager turnManager,
        bool isSoulBurn = false)
    {
        var effects = isSoulBurn ? skill.soulBurnEffects : skill.effects;

        foreach (var effect in effects)
        {
            var resolvedTargets = ResolveTargets(effect.target, caster, targets, allUnits);

            foreach (var target in resolvedTargets)
            {
                switch (effect.type)
                {
                    case EffectType.Damage:
                        ResolveDamage(caster, target, effect);
                        break;

                    case EffectType.Heal:
                        int healAmt = Mathf.RoundToInt(caster.FinalATK * effect.multiplier);
                        target.Heal(healAmt);
                        break;

                    case EffectType.ApplyDebuff:
                        float landChance = effect.debuffChance;
                        if (ElementalAffinity.HasAdvantage(
                                DebuffCategory.GetDebuffElement(effect.debuffToApply),
                                target.Data.element))
                            landChance += ElementalAffinity.DebuffChanceBonus;

                        if (Random.value <= landChance)
                        {
                            var debuff = new DebuffInstance(
                                effect.debuffToApply,
                                effect.debuffDuration,
                                effect.multiplier,
                                caster);
                            target.TryApplyDebuff(debuff, caster);
                        }
                        break;

                    case EffectType.PushCR:
                        turnManager.PushCR(target, effect.crPushAmount * 100f);
                        break;

                    case EffectType.PullCR:
                        turnManager.PullCR(target, Mathf.Abs(effect.crPushAmount) * 100f);
                        break;

                    case EffectType.GrantExtraTurn:
                        // Re-queue the caster immediately at 100 CR
                        turnManager.PushCR(caster, 100f);
                        break;
                }
            }
        }

        // Generate Soul
        soulManager.AddSoul(skill.soulGenerated);

        // Set cooldown
        caster.SetCooldown(skill.slot, skill.cooldownTurns);
    }

    private static void ResolveDamage(BattleUnit caster, BattleUnit target, SkillEffect effect)
    {
        float baseDamage = caster.FinalATK * effect.multiplier;

        // Elemental affinity
        float affinity = ElementalAffinity.GetMultiplier(caster.Data.element, target.Data.element);
        baseDamage *= affinity;

        // Crit check
        bool isCrit = Random.value <= caster.FinalCritRate;
        if (isCrit) baseDamage *= caster.FinalCritDamage;

        // Weaken debuff on target reduces their damage dealt — here we check attacker buffs
        // (target's Weaken reduces their own outgoing damage, handled in BattleUnit)

        target.TakeDamage(Mathf.RoundToInt(baseDamage));
    }

    private static List<BattleUnit> ResolveTargets(
        TargetType targetType,
        BattleUnit caster,
        List<BattleUnit> selectedTargets,
        List<BattleUnit> allUnits)
    {
        bool isPlayer = caster.IsPlayerTeam;
        return targetType switch
        {
            TargetType.SingleEnemy  => selectedTargets.Take(1).ToList(),
            TargetType.AllEnemies   => allUnits.Where(u => u.IsPlayerTeam != isPlayer && !u.IsDead).ToList(),
            TargetType.SingleAlly   => selectedTargets.Where(u => u.IsPlayerTeam == isPlayer).Take(1).ToList(),
            TargetType.AllAllies    => allUnits.Where(u => u.IsPlayerTeam == isPlayer && !u.IsDead).ToList(),
            TargetType.Self         => new List<BattleUnit> { caster },
            _ => new List<BattleUnit>()
        };
    }
}
```

```csharp
using UnityEngine;

public class SoulManager
{
    public int CurrentSoul { get; private set; }
    public const int MaxSoul = 10;

    public event System.Action<int> OnSoulChanged;

    public void AddSoul(int amount)
    {
        CurrentSoul = Mathf.Min(MaxSoul, CurrentSoul + amount);
        OnSoulChanged?.Invoke(CurrentSoul);
    }

    public bool TrySpendSoul(int cost)
    {
        if (CurrentSoul < cost) return false;
        CurrentSoul -= cost;
        OnSoulChanged?.Invoke(CurrentSoul);
        return true;
    }

    public bool CanSoulBurn(SkillData skill) =>
        skill.hasSoulBurn && CurrentSoul >= skill.soulBurnCost;
}
```

## Phase 7: Dual Attack

### DualAttackResolver.cs
```csharp
using System.Collections.Generic;
using System.Linq;
using UnityEngine;

public class DualAttackResolver
{
    private const float BaseDualAttackChance = 0.05f;

    // Called after any S1 basic attack lands
    public BattleUnit TryTriggerDualAttack(
        BattleUnit attacker,
        List<BattleUnit> allies,
        List<BattleUnit> enemies)
    {
        float chance = BaseDualAttackChance;
        // TODO: add artifact/passive bonuses to chance here

        if (Random.value > chance) return null;

        // Pick a random living ally (not the attacker)
        var eligibleAllies = allies
            .Where(a => !a.IsDead && a != attacker)
            .ToList();

        if (eligibleAllies.Count == 0) return null;

        var dualAttacker = eligibleAllies[Random.Range(0, eligibleAllies.Count)];

        // The dual attacker uses their S1 at no cooldown cost, no soul gen
        var s1 = dualAttacker.Data.s1_BasicAttack;
        var target = enemies.Where(e => !e.IsDead).OrderBy(_ => Random.value).FirstOrDefault();
        if (target == null) return null;

        // Execute S1 directly — no soul gen, no cooldown, no extra dual trigger
        SkillResolver.Execute(
            dualAttacker, s1,
            new List<BattleUnit> { target },
            enemies,
            new SoulManager(), // throwaway — no soul from dual attacks
            null,              // no turn manipulation from dual attacks
            isSoulBurn: false);

        return dualAttacker;
    }
}
```

## Phase 8: Battle Manager (Orchestrator)

The `BattleManager` drives the entire combat loop. It sequences turns, handles player input, routes skill execution, and checks win/loss conditions.

### BattleManager.cs
```csharp
using System.Collections;
using System.Collections.Generic;
using System.Linq;
using UnityEngine;

public class BattleManager : MonoBehaviour
{
    [Header("Setup")]
    public HeroData[] playerHeroes;   // 4 heroes
    public HeroData[] enemyHeroes;    // up to 4 enemies

    private List<BattleUnit> playerUnits = new();
    private List<BattleUnit> enemyUnits  = new();
    private List<BattleUnit> allUnits    = new();

    private TurnManager     turnManager;
    private SoulManager     soulManager;
    private DualAttackResolver dualAttack;

    private BattleUnit activeUnit;

    private void Start()
    {
        InitBattle();
        StartCoroutine(BattleLoop());
    }

    private void InitBattle()
    {
        foreach (var data in playerHeroes)
        {
            var unit = new BattleUnit(data, isPlayerTeam: true);
            playerUnits.Add(unit);
            allUnits.Add(unit);
        }
        foreach (var data in enemyHeroes)
        {
            var unit = new BattleUnit(data, isPlayerTeam: false);
            enemyUnits.Add(unit);
            allUnits.Add(unit);
        }

        turnManager = new TurnManager(allUnits);
        soulManager = new SoulManager();
        dualAttack  = new DualAttackResolver();
    }

    private IEnumerator BattleLoop()
    {
        while (!IsBattleOver())
        {
            activeUnit = turnManager.AdvanceToNextTurn();

            if (activeUnit.IsDead) continue;

            activeUnit.OnTurnStart();

            if (activeUnit.IsStunned)
            {
                yield return new WaitForSeconds(0.5f); // brief pause for stunned animation
                continue;
            }

            if (activeUnit.IsPlayerTeam)
                yield return StartCoroutine(HandlePlayerTurn(activeUnit));
            else
                yield return StartCoroutine(HandleEnemyTurn(activeUnit));

            // Check dual attack after S1
            if (lastSkillUsedWasS1)
            {
                var dualAttacker = dualAttack.TryTriggerDualAttack(
                    activeUnit,
                    activeUnit.IsPlayerTeam ? playerUnits : enemyUnits,
                    activeUnit.IsPlayerTeam ? enemyUnits  : playerUnits);

                if (dualAttacker != null)
                    Debug.Log($"{dualAttacker.Data.heroName} joins with a Dual Attack!");
            }

            CleanUpDead();
        }

        EndBattle(playerUnits.Any(u => !u.IsDead));
    }

    private bool lastSkillUsedWasS1;

    private IEnumerator HandlePlayerTurn(BattleUnit unit)
    {
        // Signal UI to enable player input
        // BattleHUD.Instance.ShowSkillButtons(unit, soulManager);
        yield return new WaitUntil(() => playerActionSubmitted);
        playerActionSubmitted = false;
    }

    // Called by UI when player taps a skill button
    private bool playerActionSubmitted;
    public void SubmitPlayerAction(SkillSlot slot, BattleUnit target, bool soulBurn = false)
    {
        var skill = slot switch
        {
            SkillSlot.S1 => activeUnit.Data.s1_BasicAttack,
            SkillSlot.S2 => activeUnit.Data.s2,
            SkillSlot.S3 => activeUnit.Data.s3_Ultimate,
            _ => null
        };

        if (skill == null || !activeUnit.IsSkillReady(slot)) return;
        if (soulBurn && !soulManager.TrySpendSoul(skill.soulBurnCost)) return;

        lastSkillUsedWasS1 = slot == SkillSlot.S1;

        SkillResolver.Execute(
            activeUnit, skill,
            new List<BattleUnit> { target },
            allUnits,
            soulManager,
            turnManager,
            soulBurn);

        playerActionSubmitted = true;
    }

    private IEnumerator HandleEnemyTurn(BattleUnit unit)
    {
        yield return new WaitForSeconds(0.8f); // simulate AI "thinking"

        // Simple AI: use highest-priority available skill, target lowest HP enemy
        var skill = PickEnemySkill(unit);
        var target = playerUnits
            .Where(u => !u.IsDead)
            .OrderBy(u => u.CurrentHP)
            .First();

        lastSkillUsedWasS1 = skill.slot == SkillSlot.S1;

        SkillResolver.Execute(
            unit, skill,
            new List<BattleUnit> { target },
            allUnits,
            soulManager,
            turnManager);

        yield return new WaitForSeconds(0.5f);
    }

    private SkillData PickEnemySkill(BattleUnit unit)
    {
        // Prefer S3, then S2, fall back to S1
        if (unit.IsSkillReady(SkillSlot.S3) && unit.Data.s3_Ultimate != null)
            return unit.Data.s3_Ultimate;
        if (unit.IsSkillReady(SkillSlot.S2) && unit.Data.s2 != null)
            return unit.Data.s2;
        return unit.Data.s1_BasicAttack;
    }

    private void CleanUpDead()
    {
        enemyUnits.RemoveAll(u => u.IsDead);
        playerUnits.RemoveAll(u => u.IsDead);
        allUnits.RemoveAll(u => u.IsDead);
    }

    private bool IsBattleOver() =>
        playerUnits.All(u => u.IsDead) || enemyUnits.All(u => u.IsDead);

    private void EndBattle(bool playerWon)
    {
        Debug.Log(playerWon ? "Victory!" : "Defeat.");
        // Trigger result screen, distribute rewards
    }
}
```

```csharp
public static class SetBonusManager
{
    public static void ApplySetBonuses(List<GearInstance> equippedGear, StatModifiers modifiers)
    {
        var setCounts = equippedGear
            .GroupBy(g => g.set)
            .ToDictionary(g => g.Key, g => g.Count());

        foreach (var (set, count) in setCounts)
        {
            if (count >= 4) Apply4PieceBonus(set, modifiers);
            else if (count >= 2) Apply2PieceBonus(set, modifiers);
        }

    private static void Apply4PieceBonus(GearSet set, StatModifiers m)
    {
        switch (set)
        {
            case GearSet.Assault:     m.AddMultiplier(StatType.ATK,         0.35f); break;
            case GearSet.Velocity:    m.AddMultiplier(StatType.Speed,        0.25f); break;
            case GearSet.Destruction: m.AddFlat(StatType.CritDamage,         0.40f); break;
            case GearSet.Lifesteal:   /* handled in damage resolution */ break;
            case GearSet.Counter:     /* handled in damage resolution */ break;
        }
    }

    {
```

## Key Design Notes

- **ScriptableObjects for all static data** — heroes, skills, gear are authored as Unity assets, not hardcoded. Makes iteration fast.
- **BattleUnit separates data from runtime state** — `HeroData` is immutable; `BattleUnit` is the mutable runtime wrapper. Never mutate `HeroData` during battle.
- **Debuffs are typed by element** — Fire debuffs (DoT) land better vs Earth, Dark debuffs (stat reduction) land better vs Light. This is enforced in `TryApplyDebuff` via `DebuffCategory.GetDebuffElement`.
- **CR is float-based, not turn-based** — do not convert to integer ticks. The float CR system is what allows smooth speed scaling and fractional manipulation.
- **Soul is a team resource** — it belongs to `BattleManager`/`SoulManager`, not to individual `BattleUnit` instances.
- **Game modes are thin wrappers** — all the real logic lives in `BattleManager`. Modes only override setup and reward logic.
